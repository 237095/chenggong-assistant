# 成都工业职业技术学院学费减免实施办法（试行）

为保证家庭经济特别困难的学生顺利完成学业，同时鼓励他们努力学习，艰苦朴素，积极进取，根据国家教育部、财政部《国家教委关于对普通高等学校经济困难学生减免学杂费有关事项的通知》教财[1995]30 号文件精神，结合我院实际特制定本办法。

**第一条** 减免范围

学院全日制在籍在校且家庭经济特别困难学生，同时学习刻苦努力，遵守校规校纪，积极参加集体活动，生活艰苦朴素、勤俭节约，并符合下列条件之一者，可申请减免全部或部分学费。

（一） 孤残学生

 孤儿需提供孤儿证明或公安部门与民政部门联合开具的父母死亡证明；残疾学生需提供本人残疾证明。

（二）其他家庭经济特别困难无法缴纳学费的学生提供乡（镇）或街道民政部门加盖公章予以确认的贫困证明；班级和系部确认的特别困难学生。

**第二条** 减免条件

（一）享受减免政策的学生需具备：

1.热爱社会主义祖国，拥护中国共产党的领导，遵守宪法和法律，遵守学校的规章制度；

2.诚实守信，道德品质优良，上学年无违纪记录；

3.在校学习期间，上一学年度没有补考或重修课程；

4.积极参加班级和学院各项活动。

（二）有以下情况之一者，不能给予减免：

1.拥有或使用高档通讯工具的；

2.购买或长期使用电脑的（专业学习需要的除外）；

3.购买或使用高档娱乐电器、高档时装或高档化妆品等奢侈品的；

4.节假日经常外出旅游的；

5.未经允许在校外租房的；

6.经常出入餐厅、酒吧、网吧等娱乐场所消费的；

7.有其他高消费或奢侈消费行为的。

**第三条** 减免程序

1.本人提交书面申请，并提交相关证明；

2.辅导员组织班级学生民主评议，符合条件者上报系；

3.系领导确认；

4.报学生工作处审核；

5**.**上报学院审批。

**第四条** 要求

（一）各系要加强对困难学生减免学费工作的领导，确保学生不因经济困难而辍学；

（二）对弄虚作假者，除全额退回减免的学费外，将视情节给予批评教育或纪律处分。

（三）对减免学费的学生，如发现有严重违纪等不良行为，除全额退回减免的费用外，将取消以后的减免资格。

**第五条** 本办法由学生工作处负责解释。

**第六条** 本办法自发布之日生效。
