# 学生请假考勤管理规定（修订）

为严格学生考勤管理，规范请假手续，特制定本办法。

**第一条** 学生必须按规定的时间到校，办理注册手续。

**第二条** 学生要按时参加教学计划规定和学校统一组织的活动。上课、实习、实训、劳动、军训、集会、社会实践，均进行考勤，因故不能参加者，必须事先请假。未经请假，请假未准或无特殊原因事后补假者及超过假期者，一律按旷课论处。对旷课学生，根据旷课时间多少，情节轻重及认错态度给予批评教育直至纪律处分。

**第三条** 上课、实习、实训、设计（论文）等由任课老师和学生班长同时负责考勤，并将缺勤情况及时报告学生所在二级学院学生学生工作办公室。一般缺勤应每周报告给学生所在二级学院，如有旷课，应立即报告学生所在二级学院。各二级学院每月公布一次考勤情况，如有旷课应立即对学生本人进行批评教育。有关领导、教务处和教师可随时抽查学生考勤情况。

**第四条** 学生因病请假，必须持有医务室（或二级甲等以上医院）证明；学生原则上不准请事假，遇特殊情况时，须事先办理书面请假手续；学生请假 1 天以内辅导员审批并存查；3 天以内，二级学院学生工作办公室主任批准；7 天以内，二级学院书记或者副书记批准；7 天以上由二级学院办公会研究。

**第五条** 学生因公请假，由有关单位提出书面报告，经学生所在二级学院批准。公假超过 7 天，送学生工作处备案。

**第六条** 考试和补考期间对病事假应严格掌握，无特殊情况均应先书面请假申请缓考，凡无正当事由后补假或手续不完全，均按旷考处理，并不得参加学期补考。

**第七条** 学生请假期满，应按请假权限及时办理销假手续；如需继续请假者，须持有关证明按规定办理续假；未销假按旷课处理。

**第八条** 上述所规定适用于正常的教学和实践活动，免修课程须事先征得主管部门及相关领导批准，且事先通知该课任课教师，方可不参与该门课考勤。

**第九条** 本规定由学生工作处、教务处负责解释。
