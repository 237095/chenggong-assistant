# 成都工业职业技术学院学生证管理规定

学生证是学生在校期间身份的证明和标志。为了加强学院学生证的管理，维护学生证使用的严肃性，制定本规定。

**第一条** 新生入学报到后，由学院统一制作学生证。学生证须加盖学院钢印、学院公章方能生效。学生证上包括学院、二级学院、专业、班级、姓名、性别、学生证号、照片等学生基本信息。

**第二条** 新生入学后，经复查符合入学条件者，准予注册，发放学生证。以后每学期注册期间，以班级为单位统一收集学生证到二级学院学生发展中心办理注册手续，加盖注册章后的学生证方为有效。学生不办理注册手 续则视为未报到。

**第三条** 凡假期需乘火车回家的学生，其学生证上的“乘车区间”一栏内的到达站名，应填写父母或其他监护人户口所在地最近的火车站名。学生往返家庭与学校区间乘坐火车时，凭附有学生“火车票减价优待证”的 学生证购票。确定后原则上不得改动。如学生家庭地址变更，应及时持本人学生证、家长工作单位人事部门或当地公安机关出具的有效证明到学生工作处更新个人信息，办理换证手续并交工本费。补办的学生证当年不享 有购票优惠待遇；学生购票优惠卡的注册时间为每年九月(以班级为单位 注 册 ) 。

**第四条 **学生证如有遗失，应及时挂失补办。程序如下：

(一)需补办学生证的学生到学生事务服务中心领取《学生证遗失补发申请表》。申请表须经辅导员或班主任审核并签署意见，二级学院学生发展中心主任审定并加盖公章。

(二)凭《学生证遗失补发申请表》到学生事务服务中心补办新学生证，并缴纳工本费。

(三)学生证办理时间为周一至周五，七个工作日后领取。

(四)补办期间如遗失学生证找回，应立即到学生工作处办理撤销挂失手续；一个人不能同时拥有两个及以上学生证。违反上述使用规定者，将视情节轻重给予批评教育，直至纪律处分，由此造成的一切后果，由本人承担。

**第五条** 学生证内容不得自行涂改。如需更改，经所在二级学院学生发展中心审核并出具证明，到学生工作处办理更改手续。凡擅自涂改或弄虚作假的，其学生证作废，并视情节给予相应纪律处分。

**第六条** 学生因降留级、复学、班级变动，由所在二级学院学生发展中心审核并出具证明，由学生本人持原证到学生工作处申请换发新证，原证遗失的须交纳工本费。

**第七条 **学生证不得作为抵押品。学生因毕业、退学、开除等原因离校时，由各二级学院学生发展中心负责收回学生证；如有遗失，应缴纳学生证挂失登报费。

**第八条** 学生证只限学生本人使用，不得转借他人使用。凡发现转借他人使用的，没收学生证并给予通报批评；因转借他人而产生的后果由学生本人负责。凡冒他人名义申请补发学生证或弄虚作假申请使用学生证的，从严处理。多次违反学生证使用规定的，给予纪律处分。

**第九条 **拾到学生证，应立即交至学生工作处，不得违规使用。

**第十条 **本规定由学生工作处负责解释。

**第十一条** 本规定自发布之日生效。
