# 成都工业职业技术学院 星级社团考评细则(试行)

 

**第一条** 星级社团评定时间为每年秋季学期11月，星级社团每年度评选 一次，学生社团所获星级不因自然年份延续而延续。

**第二条** 经成都工业职业技术学院社团评议评议委员会审核通过并且注 册满一学年、无违纪违法记录的学生社团均可参与评定。

**第三条** 对社团的星级进行评定，包括参评社团按照星级社团评定标准 所提交的材料，材料占总分值的60%,另外40%为现场答辩分数。

**第四条** 星级评定评定分数为四星级社团总分数大于等于110分；三星 级社团大于等于80分；二星级社团大于等于70分； 一星级社团大于等于 60分。五星级社团的评选，在满足四星级社团分数的基础上，须有参加市 级及以上比赛并获奖或在学院内开展了影响力较大的社团活动，由社团评 议委员会根据社团所提交的材料、现场答辩、社团影响力等从四星级社团 中评选一个。

**第五条** 星级社团评选中，参选社团只有全部满足必选项中的七项条件， 评星程序才会启动下一环节。
