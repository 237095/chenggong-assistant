# 成都工业职业技术学院

# 学生临时困难补助管理办法

# (试行)

**第一条** 根据国家有关文件精神，为切实做好家庭经济困难学生资助工作，进一步完善家庭经济特别困难学生的资助和保障机制，使他们能安心学习，顺利完成学业，结合学院实际，特制订本办法。

**第二条** 本办法的适用对象为我院全日制在校学生。

**第三条 **困难补助方式及范围

因下列原因导致经济困难，严重影响正常学习、生活的学生，可以申请临时困难补助：

(一)学生本人突患重病或遭遇意外事故，医药费用较高，家庭承担困难 ；

(二)学生家人突患重病或家庭突遭变故，学生经济来源和学业受到较大影响；

(三)学生家庭经济困难但学院未对其进行困难认定，缺少基本生活费用，影响学业；

(四)其他需要给予补助的情况。

**第四条** 补助标准临时困难补助金额根据学生的实际困难，按照以下原则由学生工作处资助中心组织评定。

(一)学生本人或直系亲属突患重病、遭遇意外事故，补助金额最高为3000元；

(二)学生家庭突遭变故，补助金额最高为2000元；

(三)其他情况下，补助金额最高为1000元；

**第五条 **困难补助的申请流程

(一)学生申请：学生填写《成都工业职业技术学院学生临时困难补助申请表》,说明申请理由、家庭经济状况以及获得各类资助等情况，向辅导员提交申请材料；

(二)辅导员对申请临时困难补助学生的情况进行核实；

(三)二级学院初审：二级学院分管领导进行初审同意后，将结果公示3个工作日，无异议后，签署意见并加盖公章；二级学院填写《成都工业职业技术学院临时困难补助申请汇总表》,连同申请材料报学生工作处 资助中心；

(四)学生工作处资助中心审核：学生工作处资助中心对申请材料进行审核，并根据学生实际情况提出补助方案；

(五)计财处审核：计财处对补助方案进行审核，并将审核后的方案予以公示5个工作日；

(六)分管院领导审批：补助方案公示无异议后，提交学院分管院领导批准。

**第六条** 相关要求

(一)临时困难补助为不定期补助，学生一旦因突发原因而造成生活上暂时性经济困难即可向学院提出申请；

(二)申请临时困难补助的学生， 一经发现情况不属实，将立即取消或追回补助，视情节轻重给予处分，并取消其今后补助资格；

(三)学生在申请临时困难补助的同时，主要应通过参加勤工助学和申请国家助学贷款来解决生活、学习费用；

(四)获得临时困难补助的学生根据补助金额的不同，需在一年内参加学院和各二级学院组织的志愿服务活动；

(五)学生有一下情况之一的不予补助：

1.学生家中发生变故不足以影响学生完成学业；

2.符合困难认定申请条件但没提出书面申请，由此造成的临时性困难；

3.生活铺张浪费，购买或使用高档物品；

4.学习不求上进，学习态度消极；

5.受到学院纪律处分；

6.同一事由，在同一学年内不可重复申请。

**第七条** 本办法由学生工作处负责解释。

**第八条** 本办法自发布之日生效。
