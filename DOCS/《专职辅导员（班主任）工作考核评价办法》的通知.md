# 专职辅导员（班主任）工作考核评价办法

​						（修订）

**第一条 目的与依据**

为建立科学合理的辅导员（班主任）工作考核指标体系，激励辅导员（班主任）不断提高政治和业务素质，推动辅导员（班主任）队伍专业化建设，根据教育部《普通高等学校辅导员队伍建设规定》（教育部令第 43 号），结合我院实际，制定本办法。

**第二条 考评对象**

专职辅导员（班主任）。

**第三条 考评原则**

坚持以辅导员（班主任）基本素质评价为前提，以辅导员（班主任）履职考核为重点，以目标管理与过程管理相结合、定性评价与定量考核相结合、年度考核与日常考核相结合为原则。

**第四条 考评办法**

（一）由辅导员（班主任）所在的二级学院组织考评。

（二）二级学院设立考评小组。考评小组人员组成：二级学院分管学生工作（副）书记、学生发展中心主任、科室、教研室、教师代表。

（三）考评采取分项评分的办法进行，由学生评价得分、教职工考评得分二部分构成。

（四）考评作为职工年度考核，与职工年度考核同步进行，考评结果提报生工作处（团委）。

**第五条 考评分值计算**

辅导员（班主任）工作考评总分值 100 分。其中：

（一）学生评价 50 分；

（二）考评小组考评 50 分。

**第六条 考评指标体系分解**

（一）学生评价办法与指标

1．学生评价分值：50 分。

2．学生评价采用问卷的形式，参评学生涵盖被考评辅导员（班主任）所带的每个班级，参评学生人数不低于所带班级人数的 10%。

3．学生对辅导员（班主任）工作评价从为人师表、爱岗敬业、工作能力及成效等方面进行评价。

4.学生对辅导员（班主任）工作评价表（见附件一）。

（二）考评小组考评办法与指标

1. 考评小组考评分值：50 分。

2.考评分为辅导员（班主任）个人年度述职、问卷评价两部分。

3．考评小组对辅导员（班主任）工作考核评价表（见附表二）。

**第七条 考核等级评定**

考核结果分为优秀、合格、基本合格、不合格三类，评定标准为：

**（一）“优秀”的评价标准**

1．能模范履行辅导员（班主任）岗位职责，优质完成辅导员（班主任）各项工作任务。

2．积极参与并完成辅导员（班主任）“五个一”工作，并取得优良成绩，具有一定表率作用。

3.所带班级团学活动成效突出，团籍转出率名列学院前茅，“青年大学习”活动学生参学率达 90%。

4.所带班级风气良好，学生“五个一”工作取得优良成效，具有一定示范效应。

5．考核总评分值在 90 分及以上，且学生评价分值不低于45 分。

6.考评为“优秀”的占比不超辅导员（班主任）的 10%（不足一人的可四舍五入）。

（二）“合格”的评价标准

1．能较好地履行辅导员（班主任）岗位职责，完成辅导员

（班主任）各项工作任务；

2．积极参与并良好完成辅导员（班主任）“五个一”工作，

并取得一定成绩；

3.所带班级风气良好，学生“五个一”工作取得一定成效。

4．考核总评分值在 70—90 分。

（三）“基本合格”的评价标准

1．基本能履行辅导员岗位职责，但工作中还存在一定问题；

2．能完成院分院部署的各项工作，但有延误现象；

3．考核总评分值在 60—69 分之间。

（四）有下列情况之一者，为“不合格”

1．因师德师风问题受到处罚的；

2．在事关学院稳定工作中存在突出问题的；

3．所带班级发生重大责任事故，造成严重后果的；

4．考核总评分值在 60 分以下的。

**第八条 奖惩**

（一）年度考核为合格及以上的辅导员（班主任），有资格

参加年度各级各类“优秀辅导员（班主任）”荣誉称号的评选。

（二）年度考核为不合格的辅导员（班主任），责令限期改

正。连续两年考核不合格的，予以解聘。

**第九条** 本办法由学生工作处（团委）负责解释。

**第十条** 本办法于2020年9月修订后执行，原办法同时作废。
