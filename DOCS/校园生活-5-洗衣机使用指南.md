# **洗衣机使用指南**

**一** **、洗衣步骤**

1.将衣物放入洗衣机中，关好洗衣机门。

2.用手机扫码选择洗衣类型。

3.用手机支付后，等待洗衣机启动工作。

4.洗衣结束后，及时取走衣物。

**二、收费标准**

1.一 、二号公寓阳台洗衣房：单脱5分钟，收费1元；标准洗35分钟，收费4元；大件物洗45分钟，收费5元

2.一 、二号公寓盥洗室洗衣房及二号公寓楼下洗衣房

(2)滚筒式洗衣机(容量：8kg):   标准洗30分钟收费4元/次、大件洗40分钟收费5元/次

3.三号公寓洗衣房：标准洗30分钟收费3元/次，大件洗40分钟收费 4 元 / 次

**三、温馨提示**

1.自助洗衣机是同学们共用的洗衣设备，为了个人卫生安全，请不要将贴身内衣、内裤及袜子等放入洗衣机里洗涤。

2.衣服放入洗衣机之前，请检查衣服，确保衣服里没有硬币、发卡、头绳、打火机等异物。

3.洗衣前将衣服上的长带打结，纽扣扣好，拉好拉链。

4.请勿将容易掉色的衣物与其它衣服一起洗，以免染色。

5.请勿将雨衣、睡袋、羽绒服、羽绒制品、丝绸制品等防水性衣物及 贵重衣物放入洗衣机洗涤。

6.严禁将受化学品污染(稀料、汽油等)的衣物放入洗衣机洗涤。

7.洗衣机内不能放入过多衣物，衣物过多无法保证清洗干净。
