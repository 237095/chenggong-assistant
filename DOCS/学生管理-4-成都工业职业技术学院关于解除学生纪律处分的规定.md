# 成都工业职业技术学院

# 关于解除学生纪律处分的规定

 

**第一条** 为充分体现“育人为本”的教育思想，完善“教育、引导、奖励、惩戒”相结合的学生管理体系，进一步加强对受处分学生的教育和管理，帮助受处分学生改正错误，引导其积极上进，营造全校良好的学风、校风，特制定本规定。

**第二条** 解除处分适用于下列处分类型：

(一)警告

(二)严重警告

(三)记过

(四)留校察看

**第三条** 因下列行为受到纪律处分或属下列情形者，不予解除处分：

(一)反对宪法确定的基本原则，危害国家安全、破坏安定团结、扰乱社会秩序；

(二)触犯刑法、严重违反治安管理处罚法；

(三)策划、组织、参与群体性恶性事件。

**第四条** 学生受本条例第二条且不属第三条所列的纪律处分，警告、严重警告处分可以在处分决定生效6个月以后申请解除纪律处分，记过处分可以在处分决定生效9个月以后申请解除纪律处分，留校察看处分可以在 处分决定生效12个月以后(做出留校察看处分决定之日距离学生完成学业 时间不满一年的，留校察看期到学生完成学业离校之日为止)申请解除纪 律处分。

申请解除处分的基本条件：

(一)处分学生对自己原违纪行为的性质、原因、危害性有深刻的认识，并确有明显改进的表现；

(二)学生受处分后至提出解除申请前，未再受过院、系的通报批评及以上处分，积极参加二级学院帮扶活动；

(三)积极参加公益性、义务性志愿服务活动并有良好表现；

(四)表现良好，受处分后近一学年综合素质测评成绩排名班级前列。

(五)所在班级班委会讨论同意解除。

**第五条** 受纪律处分的学生具备第四条所列条件，符合以下情形之一的，可以在警告、严重警告处分决定生效3个月后，记过处分决定生效6个月、留校察看处分决定生效9个月以后申请提前解除纪律处分。

(一)受处分以后在省级以上竞赛活动中获得名次的；

(二)受处分以后在见义勇为或者保护国家、集体及公民财产和生命安全中有突出表现的；

(三)有其它突出贡献或立功表现的。

**第六条 **解除纪律处分必须经过如下程序：

(一)本人申请

受处分学生提交解除纪律处分申请书，申请书必须写明本人对原违纪行为的认识以及受处分以后的现实表现，填写《成都工业职业技术学院学生解除纪律处分审批表》(一式三份)。

(二)同学评议

受处分学生所在班召开班委会，对受处分同学能否解除纪律处分进行讨论、投票(同意要达到班委会总人数的一半为通过，未达半数则须三个月后方可重新申请解除处分)。

(三)审查及审批

辅导员签署意见，二级学院学生工作办公室审查并签署意见，二级学院书记或者副书记审批，二级学院发文。留校察看处分的解除须提交学生工作处审批；所有解除处分决定报学生工作处备案。

(四)送达

解除纪律处分决定作出后，应及时发文并送达学生本人。

**第七条** 申请解除纪律处分的学生在审查和报批过程中又违纪的，终止解除纪律处分的审查和审批。

**第八条** 本规定由学生工作处负责解释。

**第九条** 本规定自发布之日起生效。
