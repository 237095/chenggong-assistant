# **学生寝室直饮水使用指南**

**一、寝室内直饮水收费标准**

寝室内直饮水采用按学年收费模式(每年9月至第二年7月，跟机器 显示的时间天数无关，假期留校可申请免费使用),收费标准为180元/ 人/学年。

**二、过道自助直饮水收费标准**

采用按流量收费模式：收费标准为0.5元/L, 同学购买预充值卡后按 使用流量计费。直饮水卡售价为60元/张(其中卡押金10元，水费50元， 水费不退，押金可退)。温馨提示：因该卡为无记名卡，不挂失、不补办， 请同学们妥善保管。

**三、收费方式**

寝室内和过道自助直饮水一律采取手机扫二维码支付缴费的方式，直 饮水机的机身上都有缴费二维码。

**四** **、注意事项**

1.收费时间：请在新生报到的前5天手机扫二维码集中缴费。没有全 体缴费的寝室，先由寝室成员进行协商，不能达成全体使用共识的寝室， 核实并征求意见后退费，直饮水服务公司自行关机；对于不选择使用寝室 内直饮水的寝室，开学体验期结束后将由直饮水服务公司自行关机。

2.中途变更寝室的同学必须在一周内报直饮水服务公司备案，否则不 予退费。直饮水服务公司将按照实际使用时间收费、退费。
