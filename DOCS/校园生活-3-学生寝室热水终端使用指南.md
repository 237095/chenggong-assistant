# **学生寝室热水终端使用指南**

一 、趣智校园APP 用水流程：

下载趣智校园APP-- 打开趣智校园APP-- 点击“公寓洗澡”--点击扫码洗澡--扫描水表上的二维码(或点击搜索洗澡--找到对应的设备)--点击绑定1--设备显示预扣金额，同时开阀--点击“结束洗澡”-- 电磁阀关闭，停止扣费并自动生成消费帐单。

二 、热水终端按流量计费，每次用水后按照出水量进行结算。

三、使用热水时请适当调节阀门大小，控制水量，避免浪费。

四 、洗澡时可提前扫码打开热水表，热水表开阀默认待机时间5分钟， 超过5分钟未检测到水流通过自动关闭，避免忘记关阀后资金损失。

五、使用前请确保手机网络开启(WIFI 或 4G 均可)。如点击扫码洗澡 提示“设备不可用，是否切换为蓝牙模式”点击“是”(手机蓝牙自行打开)

六、洗澡时小心地滑，注意安全。

七、如室内外热水设施，管路、终端水表、水阀等出现故障，请拨打报修电话。若未解决请拨打投诉电话。
