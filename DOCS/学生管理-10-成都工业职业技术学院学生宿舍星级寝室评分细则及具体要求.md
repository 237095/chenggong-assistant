# **成都工业职业技术学院**

# **学生宿舍星级寝室评分细则及具体要求**

​						**(试行)**

 

结合《成工职院学生宿舍星级寝室评比办法》实施中的具体情况，对评比考核中的具体情况说明如下：

**(一)清洁干净(35分)**

1.室内清爽、无异味(3分);

2.床单枕头(巾)被子干净整洁(6分);

3.地面(寝室、阳台、卫生间)清洁无垃圾积水污渍，床下无积尘(8分);

4.门窗、书桌、柜子、电扇明亮无灰尘(8分);

5.墙壁干净无污迹蛛网钉，无不健康的画报等(2分);

6.洗漱台无污渍、便池无积垢(5分);

7.垃圾篓内垃圾过半未清倒(2分);

8.公寓公区卫生清扫与保洁(根据实际情况扣0-3分)

**(二)整齐规范(25)**

9.书架整齐不杂乱，桌面清爽物品少且整齐(8分)

10.床上(8分):被子折叠不规范(2分/床);床单不平整(2分/床); 床单角悬吊，垫絮未压好(2分/床);床上乱放物品(2分/床)。

11.床下鞋子未摆在一条线(4分)。

12.凳子人离开后靠近书桌摆放(2分)。

13.除以上物品外的个人物品及公用物品脏乱差(3分)。

**(三)爱寝好学(15分)**

14.寝室有寝室长及寝室长职责、寝室卫生值日制度及值日表、寝室内有绿植(少 一 项扣1分)。

15.寝室内学习氛围浓厚(发现学生有玩游戏现象，每人扣5分；未按时起床者每人扣5分))。

**(四)文明自律(25分)**

16.按时就寝，就寝纪律良好(晚归一次扣5分，夜不归宿一次扣10分，就寝时喧闹，影响他人休息一次扣5分)。

17.私拉乱接电线扣20分/人次；违规使用电器(含伪劣电器)扣20分/人次；

18.寝室离人后插线板、饮水机、电脑等电源未关闭10分/个；

19.个人电脑等贵重物品未存放到抽屉、柜子等上锁空间10分/个；

20.寝室存放酒瓶扣5分/个，饮酒扣20分。

21.其余违反宿舍管理规定的行为扣5至20分/人次。(此项扣分为0 时可直接扣总分。)

**(五)有下列违反宿舍管理规定情况之一的，当天评分为0分**

22.私自调换寝室的(双方寝室均处理);未经审批，擅自留宿他人的；

23.寝室成员有聚众闹事、参与打架斗殴者；

24.寝室成员有参与赌博者；

25.在寝室内饲养宠物的；

26.被子未叠或者基本没有打扫整理；

27.封闭门上小窗；

28.寝室内有人吸烟、或者烟头烟盒等。

## 附2星级寝室具体要求

为创造整洁、安静、文明、安全的学习和生活环境，建设温馨的学生之家，培养同学们生活自理能力，养成良好的生活习惯，学会与人相处、交流；促进学生公寓服务管理工作，特制定星级寝室要求：

**一、寝室卫生清洁要求**

1.室内清爽、无异味；

2.床单枕头(巾)、被子干净整洁；

3.地面(寝室、阳台、卫生间)清洁，无积水、污渍，床下无积尘；

4.门窗、书桌、柜子、电扇明亮无灰尘。

5.墙壁干净无污迹、蛛网、钉，无不健康的画报等

6.洗漱台无污渍、便池无积垢。

**二、寝室内务规范要求**

(一)床面整洁、平展无杂物

7.床单不平整，床单角无悬吊，垫絮无暴露。

8.被子折叠方正，放在靠窗一侧，叠口朝门的方向摆放。枕头放在另一侧 。

9.室内除蚊帐外不能晾晒悬挂任何衣物，蚊帐悬挂平整，帐幔必须用2-4个夹子夹整齐。



![img](./images/wps9.png)![img](./images/wps10.png)![img](./images/wps11.png) 

( 四 ) 阳 台

13.漱口杯摆放在洗漱台物品格的第二层：牙膏倒立、牙刷刷毛向上、漱口盅手柄向外，整齐挨着摆放成一条直线；其它沐浴露、洗发水、香皂、洗衣粉等整齐的摆放在洗漱台下边。

14.毛巾等物品要统一挂放，做到整齐美观。

15.衣物的晾晒：洗漱台上杆子晾裤子、另外一根挂衣服

( 五 ) 其 它

16.行李箱放在柜子上面；

17.窗框上面不能晾晒袜子、毛巾，窗台上不放置鞋子等。

18.清洁工具拖把放在拖把池上面，扫帚等放在卫生间内。

![img](./images/wps12.png)![img](./images/wps13.png)![img](./images/wps14.png) 

(二)床下物品摆放有序，无脏物、杂物。

10.床下鞋子从高到矮、鞋跟朝外的顺序摆以床沿灯光线为准在一条线上(或者两条直线)。

(三)书架物品摆放整齐

11.书架摆放整齐、美观，书桌台面保持干净、无杂物。

12.凳子放在书桌下。



**三、寝室文化建设要求**

19.寝室有寝室长名单、寝室值日制度及值日表(命名)

20.寝室同学团结友爱，互相关心，互相帮助，有集体荣誉感。

21.寝室全体成员学习勤奋刻苦、互帮互学

22.寝室小摆件、绿植美观大方。

 

**四** **、寝室风采建设要求**

23.遵守公寓管理制度，按时返回寝室，熄灯就寝。

24.寝室全体成员积极参加学校、班级组织的各项活动。

25.个人电脑等贵重物品存放到能够上锁的抽屉、柜子等空间。

26.寝室成员生活健康丰富，不沉迷电脑、网络和手机游戏。

27.不在寝室酗酒、赌博、吸烟及饲养宠物等。

28.无私拉乱接、不违规使用电器(含伪劣电器)。

29.寝室离人后及时关闭插线板、饮水机等电源。
