# **成都工业职业技术学院**

# **学生评优及奖励管理办法(试行)**

 

## 第一章 总 则

**第一条** 为贯彻党的教育方针，促进校风和学风建设，激励学生勤奋学习、勇于创新、积极进取，成长为德智体美劳全面发展的社会主义建设者和接班人，根据教育部《普通高等学校学生管理规定》(教育部令第41号) 《学生资助资金管理办法》(财教〔2021〕310号)《高等学校学生行为准则》规定及《成都工业职业技术学院进一步健全和完善“三全育人”工作 实施方案》精神，结合学校实际，特制定本办法。

**第二条** 学院坚持精神奖励和物质奖励相结合、以精神奖励为主的原则，对在思想品德、学业成绩、创新创业、文体活动、劳动实践、社会工作等方面表现突出的学生个人和集体给予表彰。

**第三条** 学院对获得评优及奖励的个人或集体，授予荣誉称号、颁发证书及奖金。

**第四条** 学院对各项评优及奖励的评审均遵循公开、公正、公平的原则。

**第五条** 本办法适用于我院全日制本专科学生及其构成的集体。

1. 班级成员热爱祖国，拥护中国共产党的领导，维护国家的根本制度 和国家利益，维护学校的安全稳定，无意识形态和政治安全事件发生。
2. 班委会、团支部组织机构健全，班委、班级团干部队伍配置合理， 班级工作有计划、有总结。能营造积极正能量的班级风貌，评选周期内班内无因违纪违法受到处罚的学生。
3. 班级学风浓郁，班级平均成绩居二级学院前列，班级“青年大学习” 每周参学率在该二级学院排名前10%,“宪法卫士”学习率达100%。
4. 班级学生能积极参与“义务劳动”等劳动教育相关活动、“工匠文 化节”“公寓文化节”系列活动等校级文体、公益、社会实践活动，活动参与签到率达90%,班级注册志愿者达100%,班级学生的综合素质测评良 好以上等级占比不低于60%。
5. 积极开展“星级寝室”建设，班级50%以上的寝室被评为三星及以 上“星级寝室”。
6. 班级主题班会、团课在校内具有引领、示范作用，在学院官微、学 生工作处(团委)官方新媒体平台被宣传报道至少1次。
7. 班级学生在技术技能比赛“创青春”“挑战杯”“互联网+”等比 赛中荣获市级及以上奖项，为学院赢得荣誉。
8. 班集体受到过校级及以上组织通报表扬或表彰。
9. 评选比例：以二级学院班级总数为基数，不超过2%。

(二)先进班集体评选

1. 班级成员热爱祖国，拥护中国共产党的领导，维护国家的根本制度和国家利益，维护学校的安全稳定，无意识形态和政治安全事件发生。
2. 班委会、团支部组织机构健全，班委、班级团干部队伍配置合理，班级工作有计划、有总结。能营造积极正能量的班级风貌，评选周期内班内无因违纪违法受到处罚的学生。
3. 班级学风浓郁，班级平均成绩良好，班级“青年大学习”每周参学率在该二级学院排名前20%,“宪法卫士”学习率达95%。
4. 班级学生能积极参与“义务劳动”等劳动教育相关活动、“工匠文化节”“公寓文化节”系列活动等校级文体、公益、社会实践活动，活动参与签到率达80%,班级注册志愿者达100%,班级学生的综合素质测评等 级全部合格。
5. 积极开展“星级寝室”建设，班级30%以上的寝室被评为三星及以上“星级寝室”。
6. 班集体受到过校级及以上组织通报表扬或表彰。
7. 评选比例：以二级学院班级总数为基数，不超过10%。

(三)社会实践先进集体评选

1. 团队成员热爱祖国，拥护中国共产党的领导，维护国家的根本制度和国家利益，维护学校的安全稳定，无意识形态和政治安全事件发生。
2. 按照学院对社会实践活动的指导意见和安排开展工作，团队有详细的活动方案，完成既定工作计划，形成团队实践活动报告，积极展示、交流社会实践活动成果。
3. 社会实践活动主题突出，形式新颖，内容充实，具有社会意义，取得明显成果和突出成绩，得到有关单位和社会认可，出具相关单位团队实践证明。
4. 评选比例：以社会实践团队总数为基数，不超过10%。

(四)活力团支部评选

1. 团队成员热爱祖国，拥护中国共产党的领导，维护国家的根本制度和国家利益，维护学校的安全稳定，无意识形态和政治安全事件发生。

2. 团支部班子健全，制度完善，围绕学院中心工作和团委、团总支工作安排，有计划、有检查、有总结、有创新，按规定开展组织生活，按时 收缴团费，积极完成上级团组织的工作任务，成绩突出。
3. 重视党团知识和形势政策学习，关心国家大事和学院发展，注重思 想教育，学习氛围浓厚，按时完成全部思想政治学习教育活动，“青年大学习” 活动每周参学率在二级学院排名前20%。评选周期内支部成员未受过通报批评和纪律处分。
4. 支部活动丰富，同学团结互助，经常开展团员谈心谈话活动，针对性开展工作，支部向心力和凝聚力强，群众基础好。
5. 评选比例：以二级学院团支部总数为基数，不超过10%。

(五)“以德润身”优质主题班会评选

1. 班会主题明确，有计划、有方案、有总结，切合思政要求。
2. 班会由同学自主组织，同学参与度高。
3. 班会形式有创新，效果好，在“育人”上具有显著成效，起到主题班会的示范作用，得到认可和好评。
4. 评选名额：以二级学院为单位，每个年级评选1堂。

(六)“以德润身”优秀易班工作站评选

1. 拥有完整健全的易班工作站组织架构、工作制度，有固定的办公场地。
2. 易班微社区建设内容丰富，优质推文不少于40篇，被学校易班首页采用推文不少于5篇，易班注册认证率达100%。
3. 积极配合学校易班发展中心完成易班相关的工作任务，参加学校易班发展中心召集的会议、培训。
4. 评选比例：以学院易班工作站总数为基数，不超过40%。

(七)“以德润身”优秀易班班级评选

1. 有专门的学生负责运营，班级易班注册认证率达到100%。
2. 班级易班主页内容建设丰富，信息更新及时，班级文化在易班有展示，能与党团组织生活及班团活动相结合，班级活跃度指数值排名位列年 级前10%。

3. 积极配合校易班发展中心及二级学院易班工作站举办各项活动，开展易班特色活动不少于5次，及时在易班微社区推送活动主题。
4. 评选比例：以二级学院易班共建班级总数为基数，不超过10%。

(八)“以德润身”文明寝室评选

1. 以寝室为单位参加评选，参评寝室每月均获评四星及以上“星级寝室”,且至少5次获评五星级寝室，授予“以德润身”文明寝室荣誉称号。
2. 星级寝室评定标准参见《成都工业职业技术学院学生宿舍星级寝室评比办法(试行)》。
3. 评选比例：以学院寝室总数为基数，不超过10%。

(九)“技臻至善”最强社团评选

1. 团队成员热爱祖国，拥护中国共产党的领导，维护国家的根本制度和国家利益，维护学校的安全稳定，无意识形态和政治安全事件发生。
2. 应在学院注册一学年以上，固定社员50人以上，遵守学院社团工作的相关规定，有明确的《社团章程》,组织机构健全，社团运行规范。
3. 按月开展社团活动，社团成员参与度高，活动有计划、有总结，有过程记录，资料完整，每学期至少在学院宣传阵地上发布2次活动信息。 在繁荣校园文化、服务同学成长等方面发挥积极作用，社团活动和项目在 校内外有较大的影响力。
4. 社团负责人积极参加学生社团联合会组织的各类活动和会议，评选 周期内社团成员未受过通报批评和纪律处分。
5. 评选比例：以学院注册社团总数为基数，不超过10%。

(十)“技臻至善”明星团队评选

1. 团队由我院学生自愿组成，基于共同爱好，为实现成员共同意愿而自发组织在一起的群众性组织。团队热爱祖国，拥护中国共产党的领导， 维护国家的根本制度和国家利益，维护学校的安全稳定，无意识形态和政治安全事件发生。

2. 团队在二级学院备案，接受二级学院的管理与指导，成员人数不低于3人，团队所开展的活动不能与已有的学生社团重复或相类似。
3. 评选周期内团队成员取得全部课程学分，综合素质测评全部合格，无因违反校纪法规而受到处罚的学生。
4. 积极开展志愿公益、人文艺术、创新服务、体育竞技、技术技能等类别的相关活动，活动特色鲜明，活动策划充分具体，活动实施效果良好 且具有一定的积极影响，团队活动在学院官微、学生工作处(团委)官方新媒体平台发布宣传报道信息1次及以上。
5. 评选名额：全院每年评选不超过10个。

(十一)“技臻至善”最强新媒体团队评选

1.有较完整的组织架构、固定的运营团队、明确的工作分工、稳定的信息推送；有明确的发展定位和运营特色，运营平台有较为广泛和稳定的 读者群、粉丝数，在校内外有良好的形象和影响力。

2. 积极推进传统媒体和新兴媒体融合发展，在网络舆论引导、品牌形象宣传、先进文化传播等方面推出过传播范围广、宣传实效佳的新媒体作品。
3. 评选比例：以学院备案新媒体平台总数为基数，不超过20%。

**第十条**奖学金评选

(一)学院奖学金评选

1. 热爱祖国，拥护中国共产党的领导，理想信念坚定，遵守国家法令和学院规章制度，思想健康，品行端正，团结友爱，积极向上，维护集体荣誉， 遵守社会公德。
2. 积极参加各项集体活动、社会工作，热爱劳动，讲究卫生，所在寝室获评过二星及以上“星级寝室”或个人累计志愿服务时长不少于20小时。
3. 奖学金分为一等奖学金、二等奖学金、三等奖学金，用以表彰学习目的明确，态度端正，勤奋刻苦，成绩优良的学生。
4. 一等奖学金获得者须取得全部课程学分，平均绩点不低于3,综合 素质测评等级为优；二等奖学金获得者须取得全部课程学分，平均绩点不 低于2.5,综合素质测评等级为良好及以上；三等奖学金获得者须取得全部 课程学分，平均绩点不低于2,综合素质测评等级为良好及以上。奖学金获得者需积极参加体育锻炼和文娱活动，体育达标，评选周期内未受过通 报批评和纪律处分。

5. 评选比例：以班级学生总数为基数， “ 一等奖学金”不超过2%, “二等奖学金”不超过5%, “三等奖学金”不超过8%。

(二)学院重点项目建设奖学金评选

1.评审对象为参加学院产教融合校内重点项目建设的学生。

2. 评选办法由建设管理部门根据工作需要制定。

**第十一条**校外奖励评选条件

(一)企业组织、行业协会等在学院设立的奖项，根据签订协议，在 规定范围内按相应条件和程序组织评选。

(二)上级教育主管部门下达的评选工作，按照文件规定的条件和程 序进行评选。

 

## **第四章 组织领导机构及评审程序**

第十二条 学院成立学生评优及奖励评选委员会(以下简称“委员会”)。 分管院领导任主任，宣传统战部、教务处、招生就业处、通识教育学院、二级学院、学生工作处(团委)等相关部门主要负责人为成员，指导学院 学生评优及奖励评选工作。评优及奖励评选委员会办公室(以下简称“办 公室”)设在学生工作处(团委),负责具体评优及奖励工作。

**第十三条** 奖项的推荐与审批

(一)“蜀艺成工”现代匠星、三好学生、优秀学生干部(班级、二级学院学生干部)、优秀团干部、优秀共青团员、“锲而不舍”易班先进个人、优秀毕业生、社会实践先进个人、“精益求精”团学活动先进个人、奖学 金等校内个人奖项，由二级学院根据分配的名额比例、评选条件进行初评 推荐报办公室。

(二)优秀学生干部(学生社区、社团、易班学生干部)由学生工作处(团委)按评选条件进行初评推荐后报办公室。

(三)“蜀艺成工”双创匠星、“蜀艺成工”文体匠星、“蜀艺成工”技能匠星，由二级学院按照评选条件进行初评推荐，分别报招生就业处、 通识教育学院、教务处审核后报办公室。

(四)“精益求精”优秀新媒体记者，由宣传统战部、学生工作处(团委)、二级学院按照评选条件进行初评推荐后报办公室。

(五)全优班集体、先进班集体、社会实践先进集体、活力团支部、“以德润身”优质主题班会、“以德润身”优秀易班班级、“以德润身”文明寝室、“技臻至善”最强社团、"技臻至善”明星团队校内集体奖项， 由二级学院根据分配的名额比例、评选条件进行初评推荐后报办公室。

(六)“以德润身”优秀易班工作站由学院易班发展中心初评推荐后报办公室。

(七)“技臻至善”最强新媒体团队，由宣传统战部、学生工作处(团委)、二级学院按照评选条件进行初评推荐后报办公室。

(八)企业组织、行业协会等在学院设立的奖项，由主要合作部门按照设立的奖项条件进行评选后报办公室。

(九)以上各奖项的评选情况需由二级学院及各评审单位公示5个工作日，经委员会评审后提交院长办公会审批。

**第十四条** 上级教育主管部门下达的各级各类评优工作，由评优及奖励评选委员会按照文件规定的条件和程序进行评选审定后，上报上级教育主管部门。
