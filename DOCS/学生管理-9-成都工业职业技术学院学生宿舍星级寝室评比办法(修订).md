# **成都工业职业技术学院**

# **学生宿舍星级寝室评比办法**

​						**(修订)**

 

学生宿舍是同学们学习生活的主要场所，为加强宿舍文明建设，打造清洁、整齐、文明、优雅的生活环境，养成良好的生活习惯，培养严谨的生活作风，根据国家有关规定，结合我院实际，制定星级寝室评比办法。

**一 、评比内容要求**

寝室内清洁干净无异味；物品包括行李箱、凳子、鞋子、卫生工具等摆放整齐；床面平整，不悬挂衣物。寝室长名单、寝室值日制度及值日表上墙；同学团结友爱，互相关心。遵守公寓管理制度，按时就寝；积极参加各项活动。不沉迷电脑、网络和手机游戏；不在寝室酗酒、赌博、吸烟、饲养宠物等；合理科学用电，不私拉乱接、违规使用电器。

(标准详见学生宿舍星级寝室评分细则)

 

**二 、评比办法**

公寓生活阿姨每天对寝室进行检查并公布评分，学生处、二级学院根据需要不定期抽查，由宿管中心综合评定等级。评比分数(每月平均分) 与当月寝室星级相对应：90分以上为五星级；89-85为四星级；84-80分 为三星级；79-70分二星级；69-60分为一星级；低于59分为不合格。

凡一次未打扫或三次不规范降等；两次不打扫或有严重违纪事件，取消当月星级评比资格。



**三 、奖 惩**

1.学校对五星级寝室(当月) 、连续两次四星级 (第 次月) 每个寝室给予60元奖励(一般不超过学生宿舍总数的10%)。 一学年中有五次获得五星级，其余为四星级以上的寝室，授予寝室文明建设标兵荣誉称号，每个寝室奖励400元；寝室长自动获得院级优秀寝室长称号。

2.每月第二课堂成绩测评五星级、四星级、三星级、二星级、一星级寝室成员分别加10、8、6、4、2分，室长加2分。

3.先进集体评比，班级所属寝室在学期寝室文明建设中有1/3寝室 或1/3次数未达到一星级的班级不能参加先进集体的评比。

4.毕业生离校：所属寝室在办理毕业离校手续前最近一次寝室文明建 设中未达到二星级的，暂缓办理离校手续，直到整改至达标为止。

5.将“星级寝室”评选结果与各二级学院年度学生工作考评及辅导员 评优评先挂钩，在计算各二级学院“星级寝室”创建成绩时，五星级、四星级、 三星级、二星级、一星级、无星寝室依次按100分、90分、80分、70分、 60分、40分计算，各二级学院寝室成绩=(100×五星级寝室数量+90× 四星级寝室数量+80×三星级寝室数量+70×二星级寝室数量+60×一星 级寝室数量+40×无星寝室数量)÷各二级学院寝室总数。

6.辅导员所带班级没有五星级寝室，不能参加优秀评选。

7.对于跨二级学院混合寝室，计算成绩时按权重(人数比例)计入各二级学院总分。

 

**四 、附则**

1.本办法由学生处负责解释。

2.本办法于颁布之日起实施。
