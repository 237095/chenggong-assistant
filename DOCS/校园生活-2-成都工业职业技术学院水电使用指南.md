# **成都工业职业技术学院** **水电使用指南**

 

根据《四川省高校用能指南(试行)》,《学院用能收费方案》(成 工职院党纪〔2021〕11)要求，现就水电使用收费事项公示如下：

**一、水电缴费标准**

1.教职工值班房水电费按照国家标准收费(水2.8元/吨，电0.55元/度),在校园一卡通充值使用。

2.学生公寓水电费，按照省厅标准每月每人免费定额电5度，水2吨；每学期按4.5个月核定：8人间免费充值180度电、16吨水，6人间免费使用135度电、12吨水，超额部分按国家标准缴费。

3.学院各校区学生公寓热水使用费标准0.04元/升。

**二、缴费流程及时间**

1.教师值班房用电充值。打开易校园APP, 在首页页面选择【教师公寓电费】。根据提示选择楼栋，楼层，房间，点【下一步】,选择【照明用电】, 输入金额，点【去充值】,点【立即支付】,完成充值。

2.学生公寓用电充值指南。选择【学生公寓电费】,根据提示选择楼栋，楼层，房间，点【下一步】,输入充值金额，点【充值】,点【立即支付】, 完成充值。电费充值只能使用校园卡余额进行支付。

3.故障报修：一号学生公寓楼下后勤服务中心。

4.咨询电话：13488947388。

**三、使用安全提醒**

1.安全节约用电，严禁私拉乱接；严禁使用大功率电器。

2.遵守公寓用电管理规定，离开房间拔插头，关开关，确保安全。

3.注意剩余电量，及时购电，以免因电费不足而停电带来不便。
