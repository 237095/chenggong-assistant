# **成都工业职业技术学院**

# **毕业设计(论文)管理办法**

 

为做好在校学生毕业设计(论文)管理工作，根据教育部《关于加强 普通高等学院毕业设计(论文)工作的通知》,结合我校实际情况，制定本办法。

**第一条** 毕业设计(论文)是实现高职教育培养目标的重要教学环节，在培养学生职业技能、强化职业意识、提高职业素质等方面，具有不可替代的作用。

**第二条** 毕业设计(论文)是学生必修课程，考核成绩合格是学生符合 毕业资格的条件之一。

**第三条** 学院根据人才培养方案要求，强化毕业设计(论文)过程管理，将毕业设计(论文)纳入教学计划，原则上安排在第5学期。

第四条 毕业设计(论文)教学应注重培养学生独立工作能力和创造力；培养学生具有高尚的思想品质，严谨的科学态度，虚心好学优良作风，使毕业设计(论文)达到综合训练的目的。

**第五条** 学院实行院、二级学院两级管理。教务处代表学院统筹管理学生毕业设计(论文)任务开展；各二级学院负责具体组织、实施和考核。

**第六条** 教务处职责

1.制订学院关于学生毕业设计(论文)的规定，编制教学环节需用的相关资料。

2.开展对各二级学院的毕业设计(论文)环节的教学检查与指导，组织工作经验总结与交流。

3.检查各二级学院毕业设计(论文)的档案管理情况。

4.审查毕业设计(论文)成绩。

**第七条** 二级学院职责

1.负责制订毕业设计(论文)教学工作实施方案。

2.下达毕业设计(论文)任务，审定毕业设计(论文)课题，组织学生选题，为学生配备指导教师。

3.对毕业设计(论文)的进度、质量和指导情况等进行检查，及时解决问题，确保毕业设计(论文)的教学质量。

4.在毕业答辩2周前成立答辩委员会，组织答辩工作。

5.保管毕业设计(论文)有关材料，及时上报有关资料。

**第八条** 教学文件及主要资料

1.毕业设计(论文)教学大纲是毕业设计(论文)教学的指导性文件，由专业教研室根据人才培养目标和要求编制。主要内容应包括：教学任务与基本要求、课题选择原则、成绩评定方法及依据等。

2. 毕业设计(论文)任务书是教师和学生开展毕业设计(论文)教学工作的原始依据。毕业设计(论文)任务书由教务处制定统一格式，指导教师填写具体内容。

3.毕业设计(论文)工作安排表，用来填报各专业本届毕业设计(论文)课题名称、学生安排、指导教师等情况，由各二级学院毕业设计(论文)工作小组填写并报教务处备案。

4.毕业设计(论文)教师评价表、答辩资格审查表、成绩评分表、答辩记录及评分表用来指导和规范毕业设计(论文)学生成绩评定工作，由教务处制定统一格式和项目标准。

第九条 毕业设计(论文)教学工作由各二级学院成立毕业设计(论文)工作小组具体实施，并负责审定毕业设计(论文)课题及指导教师，编报工作安排，开展工作动员，检查和控制工作进程及质量，组织答辩和成果 评比等。

**第十条** 毕业设计(论文)教学工作主要程序及规定

1.指导教师应于毕业设计(论文)开始4周前申报毕业设计(论文)课题。

2. 各二级学院毕业设计(论文)工作小组于提前两周编报毕业设计(论文)工作安排，下发毕业班，报教务处。
3. 指导教师提前2周编制完毕业设计(论文)任务书。
4. 各二级学院毕业设计(论文)工作小组需提前一周组织开展工作动员。

5. 教务处于毕业设计(论文)中期组织常规检查。
6. 毕业设计(论文)结束前两周，学院成立本届毕业设计(论文)答 辩工作领导小组，各二级学院成立答辩工作委员会。答辩工作委员会设主任1人，副主任1—2人，一般由二级学院负责人或学术水平较高的教师担任。 各专业应建立若干答辩小组，小组成员由3人以上组成，由学术水平较高的教师担任答辩小组组长并负责答辩资格主审。二级学院答辩委员会组织 领导毕业设计(论文)的答辩工作，统一答辩要求和评分标准，审定成绩，评定本届学生优秀毕业设计(论文)成果。答辩小组主持相关毕业设计(论 文)的答辩工作。

**第十一条** 毕业设计(论文)选题

以职业发展为导向，紧扣高素质技能型人才培养目标，培养学生的职 业综合能力，贴近专业岗位群，要切实做到与专业课程体系相结合，与生产实际、就业岗位、科学研究、技术开发等紧密结合，大力提倡真题真做。课题工作量要适中，与毕业设计(论文)计划时间相适应。

**第十二条** 毕业设计(论文)指导

1.毕业设计(论文)指导老师原则上应具有中级及以上专业技术职称， 具有本专业较高的业务水平，较宽的知识面，较强的实践能力和工作责任心，也可聘请企业有经验的技术人员担任。每个指导教师指导毕业设计(论文) 学生数原则上不超过10人，最多不超过15人。

2.在课题进行前，指导教师应制订详细的工作计划，认真填写好任务书。

3.在课题进程中，要适当、及时地指导学生撰写开题报告，保证设计(论文)进程的顺利进行。

4.指导教师要加强诚信和学风教育，坚决杜绝毕业设计(论文)中出 现的抄袭和弄虚作假行为。

5.指导教师要对学生的设计(论文)过程和设计(论文)结果进行及 时审查，并于毕业设计(论文)答辩开始前一周收缴审阅学生的设计(论文) 成果(文档、图纸、实物、光盘等),填写毕业设计(论文)答辩资格审查表， 给出设计(论文)评价及成绩， 一并送交本专业毕业设计(论文)答辩小组。



**第十三条** 毕业设计(论文)答辩

1.毕业设计(论文)必须进行答辩。

2.毕业设计(论文)答辩工作包括答辩资格审查、设计(论文)成果评审、课题答辩、综合评议等程序。

3.答辩要求

(1)各专业毕业设计(论文)答辩小组应由3人以上组成，其中有1人为答辩小组组长，指导教师不得作为答辩小组组长。

(2)答辩小组组长应对课题成果认真审阅，确定毕业生的答辩资格，并填写答辩资格审查表。

(3)已完成的毕业设计(论文)需经指导教师同意并通过答辩资格审 查的学生方可参加答辩，未完成毕业设计(论文)的学生或评定成绩不及格的学生，不能参加答辩。

(4)答辩时间以15—30分钟为宜，其中学生自述5—10分钟，问答10—20分钟。

**第十四条** 成绩评定

1.学生毕业设计(论文)成绩由三部分组成：平时成绩(占30%)、毕业设计(论文)成绩(占40%)和答辩成绩(占30%)。采用五级计分制：优秀、良好、中等、及格、不及格。

2.毕业设计(论文)优秀比例原则上不超过20%,且答辩成绩必须优秀。 凡参加公开评比获各级科技成果奖的课题，或在相关专业杂志上公开发表 (含录用)论文，或有制作类实物成果的，成绩可优先认定为优秀。

3.未能按时完成毕业设计(论文)或指导教师评定成绩不及格者，待完成全部设计(论文)任务后，另行安排一次补答辩时间。

4.有下列情况之一者，毕业设计(论文)成绩可直接记为不及格：

(1)未参加规定毕业设计(论文)时间超过三分之一以上者(指导老师界定);

(2)至毕业设计(论文)答辩前一周，未完成设计(论文)工作量的 二分之一者(指导老师界定);

(3)设计(论文)成果明显抄袭，且未能理解和掌握课题主要内容者(答 辩小组界定);

(4)无故不参加答辩或毕业设计(论文)工作中严重违反纪律者(答 辩小组或指导老师界定)。

**第十五条** 资料管理

1.毕业设计(论文)工作资料包括毕业设计(论文)课题审批表、工 作安排表、任务书、答辩资格审查表、教师评价表、答辩记录及评分表。 学生毕业设计(论文)成果资料包括开题报告、论文、图纸、说明书、实 物照片、光盘、产品鉴定书和图片资料、项目鉴定或验收报告等。

2.毕业设计(论文)资料由二级学院按年度分专业集中保管，成果资 料应分类造册、并签字盖章后存档，其中电子文档按班级刻光盘，每个学 生以"10位学号+姓名”建立文件夹，光盘上贴上标签，注明“xx 届xx 专业xx 班毕业设计资料”。若学生电子文档中包含有需要应用软件打开的 图纸、程序等内容时，须在该学生的文件夹内附上附件(Word 文档)注明 所需的软件名称及版本号。

3.所有资料必须批阅签字(盖章)后归档。

**第十六条** 本规定自公布之日起试行，由教务处负责解释。原《成都 业职业技术学院毕业综合实践管理规定(试行)》同时废止。
