# 全日制高职学生转专业实施细则（修订）

为贯彻落实“以人为本、以学生为中心”教学治理理念，充分尊重学生自主选择和发展诉求，保障学生合法权益，维护学校正常教学秩序，根据教育部《普通高等学校学生管理规定》（教育部令第 41 号）、《成都工业职业技术学院章程》和《成都工业职业技术学院学籍管理规定（试行）》，结合学院实际，制定本实施细则。

**第一条** 学校按照“公平、公正、公开”原则组织开展转专业工作，校内全日制高职学生均可有一次转专业机会。

**第二条** 校内转专业分为同级转专业和编入低年级转专业两种方式。其中，三年全日制高职专科（含统招、高职单招、对口高职）同级转专业在大一上学期期末、下学期开学前进行；编入低年级转专业在大一下学期期末、大二上学期开学前进行。五年一贯制高职专科同级转专业在学籍转入我院的第一学年上学期期末、下学期开学前进行；编入低年级转专业在学籍转入我院的第一学年下学期期末、第二学年上学期开学前进行。应征入伍及参加创新创业的学生转专业时间不受此限制。

**第三条** 为维护正常教学秩序，转专业要以现有教学条件和教学资源的正常运行为前提。同级转专业时，学院对提出转专业申请的学生不规定名额，但对转入某专业的人数实行宏观调控，不得超过该专业当年实际招生人数的 10%。

**第四条** 学生有下列情形之一，可提出转专业申请：

（一）学生在学习期间对其他专业有兴趣和专长，并在学校认可的竞赛中获奖，或者自主创新创业取得相关专业的成果、证书，或者获得专利授权等能够充分说明专长，可申请转入竞赛或创新创业所属领域的相关专业；

（二）有某种疾病或生理缺陷（不含隐瞒既往病史入学者），经三级甲等医院检查证明确实不宜在原专业学习，但尚能在拟转入专业学习者（需提交三甲医院的诊断证明）；

（三）确有特殊困难或其他正当原因，无法继续在本专业学习的学生，可申请转专业学习；

（四）学院根据社会对人才需求情况的变化或学院专业发展需要，专业调整或停办，保留入学资格或保留学籍并重新办理入学、复学手续的学生，经学生本人同意，由学院根据实际情况，允许转到其他相关专业就读；

（五）休学创业或退役复学后的学生，因自身情况需要转专业的，学院予以优先考虑。

**第五条** 学生有下列情形之一，不得转专业：

（一）按照四川省扶贫计划录取的；

（二）通过特殊招生形式录取的，如保送生、定向就业、艺术类、体育类、高水平艺术团、高水平运动队等；

（三）由低学历层次转为高学历层次的；

（四）依据学院学籍管理规定，已取消学籍或应予退学者；

（五）休学或保留学籍期间的学生不允许转专业；

（六）在校期间已有一次转专业记录者；

（七）不符合拟转入专业招生录取要求者；

（八）国家教育部、四川省教育厅、四川省教育考试院等部门有其他明确规定不能转专业者。

**第六条** 根据社会对人才需求情况的变化或学校专业发展需要，经学生同意，学院可以调整部分学生的专业。

====

**第七条** 办理程序。

（一）同级转专业，按以下流程办理：

1.发布通知。由教务处（科研处）按规定发布转专业通知，组织各二级学院开展转专业工作。

2.确定专业及人数。二级学院根据学院转专业人数比例要求，结合实际确定可接纳转专业学生的专业、人数，明确对拟转入学生的考核方式，制定考核方案，报教务处（科研处）备案、审核后公示。

3.学生申请。拟转专业的学生向所在二级学院提出书面申请，填写《成都工业职业技术学院学生转专业申请表》（见附件），并附家长（或监护人）的书面意见，学生所在班级辅导员进行资格预审，转出二级学院负责人签署意见，报教务处（科研处）审核、备案。

4.二级学院考核审查。转入二级学院成立评审小组，按照考核方案，对拟转入学生的思想品德、专业素质等进行综合审查与评价，签署意见，报教务处（科研处）备案。

5.教务处（科研处）审核。教务处（科研处）对二级学院拟转专业的学生进行审核，并进行不少于 5 个工作日公示，公示无异议后报分管院领导审批。

6.办理手续。通过审批的学生在规定期限内按照流程办理转专业相关手续，超过规定期限或者不按规定流程办理有关手续者，取消其转专业资格。

（二）编入低年级转专业，由学生在规定时间内提出申请，经二级学院考核和教务处（科研处）审查合格并公示，报分管院领导审批通过后办理相关手续。

**第八条** 学院对转专业的学生按照转入专业的培养计划进行学籍管理，转专业学生按转入专业的标准交纳学费。

**第九条** 学生转专业后，必须按照转入专业的人才培养方案要求，修完全部课程，满足毕业要求后方能毕业。

**第十条** 二级学院审核转专业学生已修的课程，按以下原则确认学分。

（一）与转入专业同档或高一档的通识基础课、学科基础课和专业基础课程可直接认定学分，无需重修；转入专业同级学生已修，而转入学生尚未修的学科基础课和专业课须重修。

（二）学生转专业前获得的课程学分与转入专业培养计划无关的，或者低于转入专业的通识基础课、学科基础课和专业基础课程可计算为选修课学分。

**第十一条** 本规定自颁布之日起施行，由学院授权教务处（科研处）负责解释。原《成都工业职业技术学院全日制高职学生转专业实施细则（实行）》（成工职院﹝2017﹞34 号）同时废止。
