# 学生综合素质培养管理办法（修订）

为全面贯彻党的教育方针，切实落实“以德润身、技臻至善”校训，更加具有针对性和时效性地开展学生教育工作，促进学生综合素质提升。结合我院“办一流高职、育大国工匠”的办学定位以及学院学生综合素质培养管理方案，特制定本办法。

**第一条** 学生综合素质培养管理目标

学生综合素质培养管理体系由理想信念、身心健康、学习能力、人文艺术、法纪意识、社会能力、职业素养、专业发展八大模块构成，引导学生全方位地参与学校素质培养工程，通过对学生自主规划，自主提高的过程评价，提升学生综合素质，适应学生职业能力发展和个性发展需求。

**第二条** 学生综合素质培养目标及考核内容

（一）理想信念

涵盖参加主题（纪念日）、思政教育及“三全育人”主题教育，升旗仪式，参加党校、团校学习，积极争取进步。

（二）身心健康

涵盖积极参加体育锻炼和各种体育比赛，参加心理健康教育与活动、保持良好乐观心态；遇见师长礼貌招呼，无礼仪失范且不听劝阻行为。

（三）学习能力

涵盖学习成效，名著阅读，参加学术报告、讲座，通用技能培训。

（四）人文艺术

涵盖参加工匠文化节、学生社团活动、写作、艺术活动、学科竞赛。

（五）法纪意识

涵盖遵规守纪、诚信记录。

（六）社会能力

涵盖文明寝室、劳动实践、志愿者活动。

（七）职业素养

涵盖创新创业、职业生涯规划、企业实践。

（八）专业发展

各二级学院认定。

**第三条** 学生综合素质培养管理测评形式及程序

（一）测评形式

1.采取学生申请、班级管理员初审、辅导员复审、二级学院管理中心终审、学生处全程监督的形式进行。

2.考评周期:根据规划内容周期为学月/学期/学年

（二）方法与工作程序

1.学生申请

学生根据综合素质培养管理模块内容进行网上申请。

2.班级管理员初审

各班管理员根据相关资料逐一审核学生得分，不符合规定的予以退回并写明原因，做好记录。

3.辅导员复审

辅导员负责复审学生得分，在审核过程中必须公正、严谨。

4.二级学院管理中心负责终审，终审以批量审核通过为主。

5.学生处全程监督

每月抽查二级学院 1 个班级相关资料。如发现学生弄虚作假，取消年度评优资格；辅导员弄虚作假，按照学院二级教学事故处理。

**第四条** 学生综合素质培养评测结果及应用

（一）测评成绩作为评定三好学生、优秀学生干部、优秀毕业生等荣誉称号和评定各类奖学金、助学金，以及学生推优入党、推荐就业、毕业生方案材料填写等学生情况鉴定工作的重要依据。

（二）以学期为考核周期，测评分达 300 分的学生为合格，

不合格者，需在下学期前三个月通过义工或志愿者服务等方式补修得分（10 分一工时）；三年总分达 1500 分以上的学生，将获得成都工业职业技术学院学生综合素质等级证书，其中 30%优秀者为 A 级证书，30%良好者为 B 级证书，其余为 C 级证书。

（三）本办法适用于我院全日制全体学生。

**第五条** 学生记分申请办法

（一）按时申请：收到班级管理员通知时，学生应及时申请加分，并在规定时间内申请完成；申请记分时间为每月 18 日——24 日，考核周期为上月 18 日—本月 17 日；申请前应仔细认真阅读每月申请项目并准备好证明材料，特别注意项目对应的板块及分值。

（二）写明申请理由：为更好的配合管理员初审，申请说明材料应详细完整:时间、地点、主题（活动的名称）、身份（观众或表演者）、申请分值。

（三）不得重复申请。一个项目只能申请一次（注:不要因为网络原因，一直点击申请，会造成同样的申请出现多次；辅导员批量申请后一点告诉同学哪些条目已经申请，同学不必再次进行申请记分）。

（四）申请的项目必须逐一申请，不得合并申请（比如运动会项目，参加一项申请一项的分值）。

（五）若被班级管理员初审退回，应及时补齐证明材料重新申请，并告知管理员重新审核。

（六）在班级管理员初审、辅导员复审和二级学院管理中心终审后，应再次查询审核情况，若发现非本人原因出错，应报给管理员汇总错误分值汇总表，及时上报给中心。

注:因个人原因申请不合乎规范出现错误，出现申请模块错误、申请分值错误、申请内容不详细或者过时申请的，应在管理员初审的时候解决，终审时中心对不符合规定的申请，一律退回。

**第六条** 本规定由学生工作处负责解释

**第七条** 本规定自发布之日生效
